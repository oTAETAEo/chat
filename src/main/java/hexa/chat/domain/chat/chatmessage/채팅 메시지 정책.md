# 채팅 메시지 정책 (ChatMessage Policy)

## 메시지 삭제 정책
- 사용자가 메시지 삭제를 요청할 경우, 즉시 데이터베이스에서 물리적으로 삭제하지 않습니다.
- 삭제 요청된 메시지는 '삭제 상태'로 표시되며, 사용자의 화면에서는 더 이상 보이지 않게 처리됩니다.
- 삭제 상태로 전환된 메시지는 보관 기간(30일) 동안 데이터베이스에 유지됩니다.
- 삭제 날짜로 부터 30일이 경과한 메시지는 시스템에 의해 데이터베이스에서 영구적으로 삭제(Hard Delete)됩니다.

## 정책 상세
1. **삭제 요청 시점**: 메시지의 `softDeleteInfo` 필드를 업데이트한다. 
   1. `deleted` : `ture`
   2. `deletedAt` : 시간을 기록
   3. `deletedByMemberId` : 삭제한 회원 PK
2. **복구 가능 여부**: 삭제한 메시지는 복구 불가능.
3. **물리 삭제**: 30일 보관 정책을 통해 기준일(삭제 날짜 + 30일) 이후에 삭제된 메시지를 삭제한다.
